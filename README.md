# Pi Music Player

A custom portable music player built with:
- Raspberry Pi Zero 2 W
- Waveshare 1.3" LCD HAT (240x240)
- Joystick + 3 buttons
- Python UI
- Local MP3 playback

## Features (planned)
- Album art display
- Spotify-style now-playing screen
- Joystick navigation
- Simple menu system
- Local MP3 playback
- Configurable themes

## How to run
python3 main.py
